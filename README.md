# C Programming & OS Algorithms

This repository contains a collection of C programs written to build foundational programming logic and implement core Operating System concepts. 

## 📁 Repository Structure

The files in this repository can be broadly categorized into three main sections:
### 1. Basic Logic & Mathematics
Programs designed to understand basic syntax, conditional statements, and mathematical operations in C.
* `Addition2number.c` - Calculates the sum of two numbers.
* `areaofcircle.c` - Computes the area of a circle.
* `evenodd.c` - Checks whether a given number is even or odd.
* `greter3number.c` - Finds the greatest among three numbers.
* `leapyear.c` - Determines if a given year is a leap year.
* `rootofquadratic.c` - Finds the roots of a quadratic equation.
* `simpleintrest.c` - Calculates simple interest.
* `typeoftriangle.c` / `validtriangle.c` - Checks the validity and determines the type of a triangle.

### 2. Operating System CPU Scheduling Algorithms
Implementations of various CPU scheduling algorithms used in Operating Systems.
* `fcfs.c` / `fcfsA.c` - First-Come, First-Served (FCFS) Scheduling.
* `sjf.c` - Shortest Job First (SJF) Scheduling.
* `srtf.c` - Shortest Remaining Time First (SRTF) Scheduling.
* `rr.c` - Round Robin (RR) Scheduling.

### 3. Miscellaneous 
* `Hello.c` - Standard Hello World program.
* `digitalclock.c` - A simple digital clock implementation.
